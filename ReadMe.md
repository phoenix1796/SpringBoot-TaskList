# JWT Auth using SpringBoot

## Basic SpringBoot Java server with JWT based Bearer Token auth

### Steps to run:

use command:

```
gradle bootrun
```

To create a jar,

```
gradle assemble # It will create a jar file in {ProjectRoot}/build/libs

# Run the jar file using:
java -jar {jarName}
```
